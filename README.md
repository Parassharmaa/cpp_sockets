## Sockets Communication + Encryption

---

#### C++

**Client-Server Communication using c++**


``` bash

$ make all

$ ./server 8090

$ ./client 127.0.0.1 8090

```

---

#### Python

**Client-Server Communication using python3**

``` bash

$ python3 server.py 8090

$ python3 client.py 127.0.0.1 8090

```
